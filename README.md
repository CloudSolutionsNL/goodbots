# goodbots
Verify IP addresses of respectful crawlers like Googlebot by reverse dns and forward dns lookups.
## Supported Crawlers
* googlebot
  * .googlebot.
  * .google.
* msnbot
  * .msn.
* bingbot
  * .msn.
* pinterest
  * .pinterest.
* yandex
  * .yandex.
* baidu
  * .baidu.
* coccoc
  * .coccoc.
